# BytecodeFile Specification

一个字节码文件有以下结构:
- 文件头
- 常量池
- 函数列表
- 指令

## 文件头
固定 8 Byte
'Y' 'S' 'B' 'C' 0 0 0 0

## 常量池
由多个条目构成.

|      |       |
|:----:|:-----:|
| 总条目数 | 1Byte |
|  长度  | 1Byte |
|  内容  | NByte |
| ...  |  ...  |

使用 `LOAD_CONST` 加载

## 函数列表
由多个条目构成.

|      |       |
|:----:|:-----:|
| 总条目数 | 1Byte |
|  长度  | 1Byte |
|  参数  | NByte |
| 返回值  | NByte |
| 指令起始 | 4Byte |
| ...  |  ...  |

使用 `INVOKE & RETURN` 系列调用&返回

## 指令
由数个字节码构成.
